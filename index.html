About this web app

This is a console.log() app. Please open your console log in your web browser.

If you are using a Mac computer the keyboard shortcut is CMD + Option + J to open console.

If you are using a Windows computer the keyboard shortcut is F12 to open console.

Please use the following commands to play some Pacman

- place(num, num, string)

place() command to place Pacman on the grid in positon X,Y and facing NORTH, SOUTH, EAST or WEST

Examples of how to place Pacman on the board
  • place(0, 0, 'west')
  • place(1, 2, 'north')
  • place(0, 4, 'south')

- move()

move() command will move Pacman one space if it is a valid move. Pacman will not move if he is located on the edge of the board.

- left()

left() command will rotate Pacman 90 degrees in the specified direction without changing the position of Pacman.

- right()

right() command will rotate Pacman 90 degrees in the specified direction without changing the position of Pacman.

- report() command

report() will announce the X, Y and F of Pacman.

- board()

board() command will show the Pacman board.
